# AECRETEC

AECRETEC Enables Creative Realizations, Empowering Technology for Everyone's Creations.

（元々はAECRETEC is Everyone's CREative TEChnology.の再帰的頭字語"的"な名前の投稿サイトアイディアでした）

## プロジェクト概要

AECRETECは、全ての創作者のための分散型プラットフォームの概念提案です。この構想は、AIアシスタント（Perplexity）の支援を受けて、非プログラマによって立ち上げられました。

本プロジェクトの目的は、創作物の自由と長期的な保全を実現するための革新的なシステムを提案することです。プルーフオブスペースタイム（PoST）を採用し、デジタルコンテンツの分散保存、共有、収益化を支援する構想を提示します。

### 主な特徴（構想）

- 分散型アーキテクチャによるデータの永続的保存
- プルーフオブスペースタイム（PoST）コンセンサスメカニズム
- 多様なデジタルコンテンツのサポート（文章、イラスト、動画、音楽等）
- クリエイター主導の価格設定とライセンス管理
- Arsumトークンによる取引と報酬分配

## Arsumトークン

Arsumは、AECRETECプラットフォーム上で使用される仮想通貨トークンです。その名称は"Art"と"Sum"を組み合わせたもので、創造性の総和を表現しています。Arsumは、コンテンツの取引、クリエイターへの報酬支払い、ストレージ提供者へのインセンティブ付与など、エコシステム内での価値交換に使用されます。

## プロジェクトの目標

1. クリエイターに公平で透明性の高い収益化の機会を提供する構想を示す
2. 中央集権型プラットフォームへの依存を減らし、創作の自由を促進する方法を提案する
3. デジタルコンテンツの長期保存と可用性を保証するシステムを構想する
4. グローバルなクリエイティブコミュニティの形成を支援する枠組みを提示する

## モジュール説明

### PoSTコンセンサスメカニズム
PoST（Proof of Space-Time）は、ノードのストレージ容量と時間を考慮したコンセンサスメカニズムです。

主な機能：
- ノードの追加: `addNode(nodeId, space, time)`
- コンセンサス計算: `calculateConsensus()`

### 分散ストレージシステム
データを分散して保存するシステムです。

主な機能：
- データの保存: `store(key, value)`
- データの取得: `retrieve(key)`
- キーの一覧取得: `listKeys()`
- キーの削除: `removeKey(key)`

### Arsumトークン
プロジェクト内で使用される仮想通貨トークンです。

主な機能：
- トークンの発行: `mint(address, amount)`
- トークンの転送: `transfer(from, to, amount)`
- 残高確認: `getBalance(address)`
- 総供給量の取得: `getTotalSupply()`
